Export Instructions

  1. Open this HTML file in a browser (Safari, Chrome, or Firefox)
  2. Right-click on the icon and select "Inspect Element" or "Inspect"
  3. In the developer tools, right-click on the <svg> element
  4. Copy the SVG and save it as RxTimer-Icon.svg
  5. Use a tool to convert SVG to PNG at 1024x1024:
    • Online: CloudConvert, SVGtoPNG, or similar
    • Mac: Open in Affinity Designer, Sketch, or Figma and export
    • Command line: rsvg-convert -w 1024 -h 1024 RxTimer-Icon.svg -o AppIcon.png
  6. Alternative quick method: Take a screenshot of just the icon at high resolution, then resize to exactly 1024x1024px in Preview or Photoshop

Design Notes

  • Style: Medical-inspired "Rx" prescription symbol with timer ring
  • Colors: Deep blue gradient (#1E3A8A) with white Rx and orange accent
  • Timer Ring: Partial circular progress ring suggesting countdown/timing
  • Minimalist Clock: Subtle clock hands at 2 o'clock position (top right)
  • iOS Compliance: 226px corner radius for iOS icon standard

Alternative Design Options

If you'd like variations, you can modify the SVG above:

  • More orange: Change timer ring to be more prominent (increase opacity)
  • Different timer position: Adjust the timer arc path
  • Bolder Rx: Increase stroke-width on the Rx paths
  • Remove clock accent: Delete the clockAccent group for cleaner look
